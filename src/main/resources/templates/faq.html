Frequently Asked Questions

Quick answers about D3FENDer, assessment methods, gap detection, SBOM analysis, and defensive recommendations.

What is D3FENDer?

D3FENDer is a threat-driven security assessment platform that helps identify security gaps using MITRE ATT&CK for adversary behavior and D3FEND for defensive techniques.

What assessment methods are supported?

D3FENDer supports standard text-based assessment, guided questionnaire assessment, structured JSON input, and SBOM vulnerability analysis.

How does the standard assessment work?

The user describes their system in plain text. D3FENDer uses semantic processing to interpret the input, normalize it into defensive capabilities, evaluate relevant security rules, and generate findings.

How does the questionnaire assessment work?

The questionnaire guides the user through structured security questions. The answers are converted into normalized defensive measures, then evaluated by the rule engine to identify missing protections.

How does JSON input work?

JSON input allows users to submit structured security data directly. Since the data is already structured, D3FENDer can normalize it more directly and run it through the same rule evaluation pipeline.

How does SBOM analysis work?

The uploaded SBOM file is parsed to extract software components. D3FENDer checks those components against vulnerability data sources such as OSV, records discovered vulnerabilities, and presents the affected components to the user.

How does gap detection work?

D3FENDer normalizes the user’s defensive capabilities, retrieves matching assessment rules, checks each rule’s conditions, and creates a security finding when important defensive measures are missing.

What role does the rule engine play?

The rule engine compares normalized security capabilities against predefined rule conditions. For example, if a required protection is absent, the engine can create a finding and recommend relevant D3FEND techniques.

What are MITRE ATT&CK and D3FEND?

MITRE ATT&CK describes adversary tactics and techniques. D3FEND complements it by describing defensive techniques that can help detect, harden, isolate, deceive, or evict adversary behavior.

Is D3FENDer a replacement for a full security audit?

No. D3FENDer supports security analysis and prioritization, but findings should be reviewed by qualified security personnel before production decisions are made.
